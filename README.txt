# Cypress Automation Framework

Framework para solicitud de prueba tecnica

## Instalacion

Descargar a instalar Node js ![node](https://nodejs.org/en/download/current)
Clonar o descargar este repo

```bash
npm install
o desde la consola de visual studio code: npm install
```

## Ejecutar

```
- npx cypress open: para abrir cypress y ejecutar desde la interfaz

- Seleccionar E2E Testing
- Seleccionar navegador Chrome
- Click en Start E2E Testing in Chrome
- Seleccionar test a ejecutar

```
- npm test: para ejecutar los test y que se genere el informe

```

- En la carpeta reports se encuentra la carpeta screenshot y el reporte en html
- Dentro de cypress, testcases se encuentran los casos de pruebas manuales
